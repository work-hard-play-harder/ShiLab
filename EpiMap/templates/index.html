IEpiMap: An open machine learning web service and model repository for the epistasis analysis in diseases and complex traits

Our web server provides an easily used pipeline, through which biologists and biomedical researchers can train a specific model based on their own data or predict their new data based on a shared model in model repository, and visualize results using interactive visualizations.

The easiest way to use

Users will only need to upload their data to our server, and the results can be quickly predicted based on our models trained online.


Model repository

due to the challenges of replicating large models on large datasets, it is hard to achieve a good model following existing work. We will provide a model repository which is a set of unified containers with trained models. We will package the depending packages, configurations, model parameters, and training details in the container, which can be run on any operating system.

Fancy visulization

To help understand the results, we will provide interactive visualization of intermediate results and final results, with options to zoom in and out for diverse views of the data and results. We will provide links to metadata of omics data and external resources, such as gene annotations and biological pathways, to help interpret the results.
